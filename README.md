# Лабиринт сокровищ

Консольная текстовая приключенческая игра на Python, в которой игрок
исследует лабиринт, решает загадки, избегает ловушек и пытается добраться
до сокровищ.

---

## Описание проекта

**Лабиринт сокровищ** — это текстовая adventure-игра, запускаемая из терминала.
Игрок перемещается между комнатами, собирает предметы, использует их,
решает загадки и в конечном итоге открывает сундук с сокровищами.

Проект реализован с использованием:

- Python 3.11+
- Poetry
- модульной структуры
- линтера Ruff

---

## Установка и запуск

### 1. Клонировать репозиторий

```bash
git clone git@github.com:sashaprovorova/project1_provorova_sasha_dpo.git
cd project1_provorova_sasha_dpo
```

### 2. Установить зависимости

```bash
poetry install
```

### 3. Запуск игры

```bash
poetry run project
```

## Игровая логика

### Комнаты

Игра состоит из набора комнат, связанных между собой переходами.
Каждая комната может содержать:

- описание
- предметы
- выходы
- загадку

Доступные комнаты:

- entrance — вход в лабиринт
- hall — центральный зал
- library — библиотека с загадкой
- trap_room — комната с ловушкой
- armory — оружейная
- chamber — спальня
- hidden_kitchen — скрытая кухня
- treasure_room — комната сокровищ

---

## Управление в игре

- north / south / east / west — перемещение по комнатам
- go <direction> — перемещение по направлению
- look — осмотреть текущую комнату
- take <item> — поднять предмет
- use <item> — использовать предмет
- inventory — показать инвентарь
- solve — решить загадку в комнате
- help — показать список команд
- quit / exit — выход из игры


### Загадки

Некоторые комнаты содержат загадки, которые игрок может попытаться решить
с помощью команды `solve`.

Особенности:

- ответы проверяются **без учёта регистра**
- поддерживаются **альтернативные варианты ответа**  
  (например: `10` и `десять`)
- после успешного решения загадка исчезает
- в некоторых комнатах за решение выдаётся награда

---

### Награды

За решение загадок в определённых комнатах игрок может получить предметы,
необходимые для прохождения игры (например, ключ от сокровищницы).

Награда добавляется в инвентарь автоматически после правильного ответа.

---

### Ловушки

В комнате `trap_room` действует особая логика:

- при неверном ответе активируется ловушка
- игрок может:
  - потерять случайный предмет из инвентаря
  - либо проиграть игру

Это делает прохождение комнаты более рискованным.

---

### Случайные события

Во время перемещения между комнатами могут происходить случайные события:

- появление предметов
- таинственные звуки
- активация ловушки (в определённых условиях)

События зависят от количества сделанных шагов.

---

### Победа

Игра считается успешно завершённой, если игрок:

1. Получил ключ от сокровищницы
2. Открыл проход в комнату сокровищ
3. Успешно открыл сундук

После этого выводится сообщение о победе, и игра завершается.
